# Tirth Patel | Interactive Terminal Portfolio

A premium, Linux-style interactive portfolio featuring a draggable window system and a functional command-line interface.

## ✨ Key Features
- **💻 Terminal CLI**: Fully interactive command line with custom commands.
- **🪟 Windows System**: Draggable and resizable windows for different sections.
- **🚀 Boot Animation**: Immersive system startup sequence on page load.
- **✉️ Contact Form**: Secure form with strict validation linked to Formspree.
- **📱 Responsive**: Optimized for various screen sizes with a grid backdrop.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons**: FontAwesome
- **Fonts**: JetBrains Mono & Inter
- **Form Handling**: Formspree API
